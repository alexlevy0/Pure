Pure

Un gestionnaire de bases de données moderne et intuitif

À propos du projet

Pure est un gestionnaire de bases de données développé avec Wails et React-TypeScript. Cette application desktop vous permet de vous connecter à vos bases de données PostgreSQL, d'explorer leurs structures et d'exécuter des requêtes SQL avec une interface utilisateur moderne et intuitive.

Technologies utilisées

  • Wails - Framework pour créer des applications desktop avec Go et interfaces web
  • React & TypeScript - Pour une interface utilisateur réactive et typée
  • Shadcn UI - Bibliothèque de composants UI modernes et accessibles
  • Go - Pour le backend robuste et performant
  • PostgreSQL - Support pour les bases de données PostgreSQL

Fonctionnalités

Gestion des connexions

Ajoutez, testez et organisez vos connexions aux bases de données

Stockage sécurisé

Vos informations de connexion sont stockées localement de manière sécurisée

Exploration des bases de données

Parcourez facilement vos bases de données et leurs tables

Éditeur SQL

Éditeur SQL intégré avec auto-complétion pour exécuter vos requêtes

Installation

Vous pouvez télécharger les versions compilées pour votre système d'exploitation depuis la section Releases de notre dépôt GitHub.

Pour les développeurs souhaitant contribuer ou compiler l'application :

  1. Clonez le dépôt : git clone https://github.com/USERNAME/Pure.git
  2. Installez les dépendances du frontend : cd frontend && npm install
  3. Lancez l'application en mode développement : wails dev
